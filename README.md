## The Blue Alliance

The Blue Alliance is an organization that consumes all the match data published by FIRST and makes this data available
to developers via an API.

The URL for this API is [The Blue Alliance API v3](https://www.thebluealliance.com/apidocs/v3). You will need an API key
which can be copied from `./resources/js/TBAInterface.js`.
